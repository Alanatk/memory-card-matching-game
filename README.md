# Memory Card Matching Game

A responsive Memory Card Matching Game built with React and Vite for the frontend internship assignment.

## Features

- 16-card matching board on Medium mode (default)
- Random shuffle on every new game
- Move counter and live timer
- Auto flip-back for non-matching pairs
- Win screen with final moves and time
- Restart game button
- Bonus: difficulty levels, best score tracking with Local Storage, flip animations, polished responsive UI

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deploy To Vercel

1. Push this project to a GitHub repository.
2. Import the repository into Vercel.
3. Keep the default settings for a Vite app.
4. Deploy and use the generated production URL in your submission.
